Registers a new sensor for this device at CommonSense. Also connects the sensor to this device.

Parameters:
contextThe application context, used to retrieve preferences.
nameThe name of the sensor.
displayNameThe sensor's pretty display name.
descriptionThe sensor description (previously "device_type").
dataTypeThe sensor data type.
valueAn example sensor value, used to determine the data structure for JSON type sensors.
deviceType(Optional) Type of the device that holds the sensor. Set null to use the default device.
deviceUuid(Optional) UUID of the device that holds the sensor. Set null to use the default device.
Returns:
The new sensor ID at CommonSense, or null if the registration failed.
Exceptions:
JSONExceptionIn case of invalid sensor details or if the request returned unparseable response.
IOExceptionIn case of communication failure during creation of the sensor.

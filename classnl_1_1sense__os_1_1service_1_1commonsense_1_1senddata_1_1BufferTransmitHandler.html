Detailed Description

Handler for tasks that send buffered sensor data to CommonSense. Nota that this handler is re-usable: every time the handler receives a message, it gets the latest data in a Cursor and sends it to CommonSense.

Subclasses have to implement getUnsentData() and onTransmitSuccess(JSONObject) to make them work with their intended data source.
